# 최종 수정안 확정 로드맵

## 1차 MVP 목표
엑셀 기반 학구 변경 관리 업무를 웹 DB 시스템으로 전환한다.

## 포함 기능
1. 로그인/권한
2. 대시보드
3. 학구 변경 요청 관리
4. 상세 작업 관리
5. 진행상태 변경/이력 관리
6. 학교현황
7. 공지사항
8. 엑셀 업로드/다운로드
9. EPSG:5186 기반 학교 위치 저장

## 제외 기능
- 학구현황
- 지도 편집
- 학구 폴리곤 직접 수정
- GeoServer 연동
- 공간검증 자동화

## 개발 순서
1주차: DB/PostGIS + 프로젝트 생성
2주차: 로그인/권한/레이아웃
3주차: 학구 변경 요청 관리
4주차: 대시보드
5주차: 학교현황
6주차: 공지사항
7주차: 엑셀 업로드/다운로드
8주차: 보안/배포 준비

## 보안 원칙
- 비밀번호 BCrypt 암호화
- API 권한 분리
- CORS 운영 도메인 제한
- 파일 업로드 확장자/용량 제한
- 개인정보 로그 출력 금지
- DB 계정 권한 최소화
- .env/application-secret.yml Git 제외
